Using the Boomerang C Decompiler

  1. Download the decompiler from Sourceforge.
  2. Boomerang requires the libgc and libexpat dependencies. Download them thusly: 
    sudo apt-get install libgc1c2:i386 libexpat1-dev:i386 lib32stdc++6 lib32z1 lib32z1-dev
    The above represent the 32bit dependency files that might be missing if you're using 64bit.  
  3. cd to your boomerang file and use the ./boomerang <binary file name> or put it in your PATH.
